Modified srsRAN Source Code for CSI Parameter Extraction

This folder contains modified srsRAN source code aimed at extracting Channel State Information (CSI) parameters.

The modification has been applied to the function:

compute_layer_hop

located in the following file:

port_channel_estimator_average_impl.cpp

The modified function enables the extraction of CSI-related parameters during the channel estimation process.

This file is compatible with srsRAN Project version 24.04, available at the following link:

https://github.com/srsran/srsRAN_Project/releases/tag/release_24_04
